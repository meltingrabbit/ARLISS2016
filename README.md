# ARLISS2016
A Rocket Launch for International Student Satellites 2016

## これはなに？
ARLISS 2016 に東大チームで参加したときの，ローバー制御プログラム．  
なお，能代宇宙イベント2016もこれで参加している．  
  
また，WebAppはローバーが撮影した写真をGoogle Map上に表示し，  
さらにその時のローバーの諸元も表示するアプリ．
